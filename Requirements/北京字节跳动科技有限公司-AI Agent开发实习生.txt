岗位职责
团队介绍：App Infra 是字节跳动面向应用的中台部门。部门依托虚拟机、编译工具链等底层技术，结合性能度量与系统特性实现深度优化，同时融合 LLM、LRM、Agent 等前沿技术，为字节跳动旗下核心应用提供终端 + 平台一体化基建与解决方案。面向研发效率，部门结合运行上下文，打通质量体系、Coding Agent、编译工具链与规模化验证，构建面向移动研发的 Agent Harness；面向商业效能，部门借助端侧特征工程、深度学习 / 决策树模型、SLM 等技术能力，围绕搜索、推荐、广告、直播、电商等领域，打造云端协同推荐、品质体验优化等端智能解决方案。如果你热爱挑战底层技术、追求极致优化，又渴望把AI转化为千万用户感受得到的流畅与智能——欢迎加入，和我们一起打造下一代移动体验！

1、面向字节跳动客户端研发场景，搭建稳定、高效的 Agent Harness（涵盖移动运行时知识库、Skills & CLI、规模化验证体系等），并基于 trace 机制持续优化长程执行的效率与效果；
2、面向客户端研发全流程，整合 Issue Diagnosis、Coding、Review 与 Validation 等能力，构建并标准化 AI Agent 体系，推动在 Trae 等 IDE 及研发平台的规模化落地，持续提升 Agent 在真实研发场景中的稳定性、可控性与研发效率。
岗位要求
1、2027届及以后毕业，计算机、软件工程等相关专业优先；
2、具备扎实的软件工程基础与客户端研发经验，有完整的研发流程经历；
3、熟悉大模型与 AI Agent 相关技术，具备良好协作能力，能与团队协作持续优化 Agent 效果。

加分项
1、具备在 IDE 或研发平台侧推动能力落地的实践经验；
2、有 AI Coding、自动化测试或开发工具链相关项目经验者优先。
